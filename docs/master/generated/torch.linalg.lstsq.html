torch.linalg.lstsq¶

torch.linalg.lstsq(input, b, cond=None, *, driver=None)¶

-> (Tensor solution, Tensor residuals, Tensor rank, Tensor singular_values)

Computes the least squares solution to the system with a batch of matrices aaa (represented by input) and a batch of vectors or matrices bbb such that

x=argminx∥ax−b∥F,x = \text{argmin}_x \|ax - b\|_F, x=argminx​∥ax−b∥F​,

where aaa is of size (...,m,n)(..., m, n)(...,m,n) and bbb is of size (...,m,k)(..., m, k)(...,m,k) or (...,m)(..., m)(...,m) . The batch dimensions of aaa and bbb have to be broadcastable.

The returned solution solutionsolutionsolution is of shape (...,n,k)(..., n, k)(...,n,k) if bbb is of shape (...,m,k)(..., m, k)(...,m,k) , and is of shape (...,n,1)(..., n, 1)(...,n,1) if bbb is of shape (...,m)(..., m)(...,m) . The batch sizes of xxx is the broadcasted shape of the batch dimensions of aaa and bbb .

Note

The case when m<nm < nm<n is not supported on CUDA.

Parameters

  • input (Tensor) – the batch of left-hand side matrices aaa of shape (...,m,n)(..., m, n)(...,m,n) with m>0,n>0m > 0, n > 0m>0,n>0

  • b (Tensor) – the batch of righ-hand side vectors or matrices bbb of shape (...,m)(..., m)(...,m) or (...,m,k)(..., m, k)(...,m,k) with m>0,k>0m > 0, k > 0m>0,k>0

  • cond (float, optional) – used to determine the effective rank of aaa for the rank-revealing drivers (see driver). Singular values s[i]≤cond∗s[0]s[i] \le cond * s[0]s[i]≤cond∗s[0] are treated as zero. If cond is None or is smaller than zero, the machine precision based on input’s dtype is used. Default: None

  • driver (str, optional) – the name of the LAPACK/MAGMA driver that is used to compute the solution. For CPU inputs the valid values are ('gels', 'gelsy', 'gelsd, 'gelss', None). For CUDA inputs the valid values are ('gels', None). If driver == None, 'gelsy' is used for CPU inputs and 'gels' for GPU inputs. Default: None

Note

Driver 'gels' assumes only full-rank inputs, i.e. torch.matrix_rank(a) == min(m, n). Drivers 'gelsy', 'gelsd', 'gelss' are rank-revealing and hence handle rank-deficient inputs. 'gelsy' uses QR factorization with column pivoting, 'gelsd' and 'gelss' use SVD. 'gelsy' is the fastest among the rank-revealing algorithms that also handles rank-deficient inputs.

Warning

The default value for cond is subject to a potential change. It is therefore recommended to use some fixed value to avoid potential issues upon the library update.

Returns

a namedtuple (solution, residuals, rank, singular_values) containing:

  • solution (Tensor): the least squares solution

  • residuals (Tensor): if m>nm > nm>n then for full rank matrices in input the tensor encodes

    the squared residuals of the solutions, that is ∣∣input@x−b∣∣F2||\text{input} @ x - b||_F^2∣∣input@x−b∣∣F2​ . If m≤nm \le nm≤n , an empty tensor is returned instead.

  • rank (Tensor): the tensor of ranks of the matrix input with shape input.shape[:-2].

    Only computed if driver is one of ('gelsy', 'gelsd', 'gelss'), an empty tensor is returned otherwise.

  • singular_values (Tensor): the tensor of singular values

    of the matrix input with shape input.shape[:-2] + (min(m, n),). Only computed if driver is one of ('gelsd', 'gelss'), an empty tensor is returend otherwise.

Return type

(Tensor, Tensor, Tensor, Tensor)

Example:

>>> a = torch.tensor([[10, 2, 3], [3, 10, 5], [5, 6, 12]], dtype=torch.float)
>>> b = torch.tensor([[[2, 5, 1], [3, 2, 1], [5, 1, 9]],
                      [[4, 2, 9], [2, 0, 3], [2, 5, 3]]], dtype=torch.float)
>>> x = torch.linalg.lstsq(a.view(1, 3, 3), b).solution
>>> x
tensor([[[ 0.0793,  0.5345, -0.1228],
         [ 0.1125,  0.1458, -0.3517],
         [ 0.3274, -0.2123,  0.9770]],

        [[ 0.3939,  0.1023,  0.9361],
         [ 0.1074, -0.2903,  0.1189],
         [-0.0512,  0.5192, -0.1995]]])

>>> (x - a.pinverse() @ b).abs().max()
tensor(2.0862e-07)

>>> aa = a.clone().select(1, -1).zero_()
>>> xx, rank, _ = torch.linalg.lstsq(aa.view(1, 3, 3), b)
>>> rank
tensor([2])

>>> sv = torch.linalg.lstsq(a.view(1, 3, 3), b, driver='gelsd').singular_values
>>> (sv - a.svd()[1]).max().abs()
tensor(5.7220e-06)
